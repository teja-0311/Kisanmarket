# KisanMarket
Agricultural Marketplace App

KisanMarket is a full-stack mobile application designed to provide a digital marketplace for agricultural products. The platform allows users to securely browse products, manage carts, place orders, and interact with an AI assistant for guidance and support. The project aims to bridge the gap between agricultural producers and consumers using modern mobile and web technologies.

## Purpose of the Project
The goal of this project is to develop a scalable and secure agricultural marketplace application that leverages mobile technology and AI assistance to enhance accessibility, efficiency, and user experience in agricultural commerce.

## Project Status
- This project is functional.
- Future enhancements such as payment integration, admin dashboards, and production deployment.
  
## Features

- User authentication with JWT-based login and OTP verification
- Secure signup, login, forgot password, and password reset
- Product browsing with filtering options
- Shopping cart management
- Order placement and order tracking
- In-app notifications
- AI assistant for user queries and support
- RESTful backend APIs
- MongoDB-based persistent data storage

## Installation & Setup

### Prerequisites
- Node.js
- MongoDB
- Expo CLI

### Backend Setup
- cd backend
- npm install
- npm start

### Frontend Setup
- npm install
- npx expo start

### Environment Variables
- MONGO_URI=your_mongodb_connection_string
- JWT_SECRET=your_secret_key
